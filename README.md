# java-web-sbt-spring-jsp-keystore-hello-world

## Description
A springboot web app with jsp support with self-signed ssl.

#W Tech stack
- java
  - springboot
  - jsp
- sbt

## Docker stack
- hseeberger/scala-sbt:11.0.2-oraclelinux7_1.3.5_2.12.10
- openjdk:17-alpine

## To run
`sudo ./install.sh -u`
[Available at](https://localhost)

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
